# Agent Capabilities

## Supported

- Parse structured specification payloads.
- Build deterministic dependency graphs and execution batches.
- Ask targeted clarification questions.
- Pause for explicit human approval.
- Create issue drafts and publish with `gh` CLI.

## Not Supported

- Inventing new workflows or replacing Spec-Kit process.
- Publishing issues without approval.
- Non-deterministic output under identical input conditions.
